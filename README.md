# Autism AI - Sparkle Labs 🧩

Multimodal behavioral analysis system for Autism using video feature extraction (pose/gaze) and RAG-based therapy planning.

## 🚀 Getting Started

### 1. Prerequisites
- **Python 3.10+**
- **Node.js 18+**
- **PostgreSQL** (running locally on port 5432)

---

### 2. Backend Setup
The backend is a FastAPI application that handles ML inference, data storage, and AI therapy generation.

1. **Navigate to backend**:
   ```bash
   cd backend
   ```
2. **Setup Virtual Environment**:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
3. **Configure Environment**:
   Ensure `backend/.env` is set up with your `DATABASE_URL` and `OPENAI_API_KEY`.
4. **Initialize Metadata**:
   If this is the first time running, build the therapy vector database:
   ```bash
   python scripts/build_chroma_db.py
   ```
5. **Run Server**:
   ```bash
   uvicorn app.main:app --reload
   ```
   Backend will be live at: `http://localhost:8000`
   API Documentation: `http://localhost:8000/docs`

---

### 3. Frontend Setup
The frontend is a Next.js application with a premium dashboard and real-time camera processing.

1. **Navigate to frontend**:
   ```bash
   cd frontend
   ```
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Run Dev Server**:
   ```bash
   npm run dev
   ```
   Frontend will be live at: `http://localhost:3000`

---

### 4. Core Features
- **Dashboard**: `http://localhost:3000/dashboard` - Main overview with custom assets (Aster bot).
- **Inference Pipeline**: Real-time edge feature extraction via MediaPipe in the browser, sent to the backend for TCN-Autoencoder scoring.
- **AI Therapy**: RAG-based plans generated by GPT-4o-mini using ESDM clinical data.

---

### 🚨 Security Note
Credentials and API keys are stored in `backend/credentials_secret.txt` (ignored by git). **Never commit this file or the `.env` file to public repositories.**
